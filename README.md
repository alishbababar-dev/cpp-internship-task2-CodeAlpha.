# User Registration and Login System (C++)

## Overview
This project is a simple User Registration and Login System developed in C++. It demonstrates the use of file handling, functions, input validation, and basic authentication concepts.

The system allows users to create accounts, stores credentials in text files, and verifies user information during login.

## Features

- User Registration
- User Login Authentication
- Username Validation
- Password Validation
- Duplicate Username Detection
- File-Based Data Storage
- Error Handling and User Feedback

## Technologies Used

- C++
- File Handling (fstream)
- Functions
- Conditional Statements
- String Manipulation

## How It Works

### Registration
- User enters a username and password.
- Username must be at least 3 characters long.
- Password must be at least 4 characters long.
- The system checks for duplicate usernames.
- User credentials are stored in a text file.

### Login
- User enters username and password.
- The system reads stored credentials from the file.
- If credentials match, login is successful.
- Otherwise, an error message is displayed.
